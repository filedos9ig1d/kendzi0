*** Instalacja

Windows:
 1. utwórz katalog:
      C:\josm_dev\plugins
 2. ściągnij plugin Kendzi3d.jar do utworzonego katalogu: 
      http://www.openstreetmap.pl/kendzi/Kendzi3d/snapshot/*/Kendzi3d.jar
 3. usuń katalog 
      C:\josm_dev\plugins\Kendzi3d
 4. uruchom JOSM używając nowego katalogu domowego. Otwórz cmd (Windows+R) i uruchom komende:
      java -Djosm.home=c:\josm_dev\ -jar josm-tested.jar
 5. wybierz plugin w oknie konfiguracji pluginów
      Edycja>Ustawienia>Plugins(zakładka)>Kendzi3d(na liście)
 6. zrestartuj JOSM. Uruchom go tą samą komendą co poprzednio.


Linux:
	Tak samo jak w windowsie
